RepoDb - A hybrid ORM library for .NET.

It is an open-source .NET ORM that bridge the gaps between micro-ORMs and full-ORMs. It helps the developer to simplify the switch-over of when to use the BASIC and ADVANCE operations during the development.

You can use the library to work with SQL Server, SQLite, MySQL and PostgreSQL Relational Database Management Systems (RDBMS).

Supported Databases

The execute methods below supports all RDBMS data providers.

  • ExecuteQuery
  • ExecuteNonQuery
  • ExecuteScalar
  • ExecuteReader
  • ExecuteQueryMultiple

Whereas the fluent methods below only supports the SQL Server, SQLite, MySQL and PostgreSQL RDBMS data providers.

  • Query
  • Insert
  • Merge
  • Delete
  • Update
